# Homefiles: Bash Environment Customization

This folder contains my custom Bash config files and deployment script to setup the terminal environment.

## Included Files
    - `.bash_aliases`: A collection of shortcuts and functions to improve productivity.
    - `setup.sh`: Automation script that backs up existing configuration and creates a symlink to this repository.

## Setup Instructions
    - Navigate to this directory `cd ~/dotfiles/homefiles`
    - Make script executable if not `chmod +x setup.sh`
    - Run Script `./setup.sh`
    - Reload environment `source ~/.bashrc`
*Reload is prompted when script is run*
